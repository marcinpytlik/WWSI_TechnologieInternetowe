# Ćwiczenia – CSS podstawy

W tym zadaniu:
- Zmieniono kolory nagłówków i czcionki.
- Dodano marginesy i padding.
- Ustawiono tło strony (gradient).
- Użyto różnych selektorów: **element** (`body`, `h1`, `p`), **class** (`.card`, `.btn`), **id** (`#main-container`, `#topbar`).

## Pliki
- `index.html` – strona z formularzem rejestracyjnym.
- `css/styles.css` – styl z komentarzami (co i gdzie jest użyte).

Otwórz `index.html` w przeglądarce i modyfikuj `css/styles.css`, żeby poćwiczyć.

# Ściąga: HTTP
- Metody: GET (idempotentny), POST (tworzenie/akcje), PUT (zastąp), PATCH (częściowa zmiana), DELETE (usuń)
- Statusy: 200 OK, 201 Created, 204 No Content, 301/302/308 Redirect, 400 Bad Request, 401 Unauthorized, 403 Forbidden, 404 Not Found, 409 Conflict, 429 Too Many Requests, 500/502/503
- Nagłówki: `Content-Type`, `Accept`, `Cache-Control`, `Authorization`, `Set-Cookie`, `X-Content-Type-Options: nosniff`, `Strict-Transport-Security`
